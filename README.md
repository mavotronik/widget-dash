# Widget Dash

Конструктор дашбордов с несколькими экранами и виджетами: часы, дата, текст, картинка.

## Возможности

- Несколько экранов с автопереключением каждые 15 секунд
- Режим настройки: перетаскивание виджетов, смена темы, добавление экранов и виджетов
- Сохранение конфигурации в `localStorage`

## Запуск

```bash
npm install
npm run dev
```

Откройте адрес, который покажет Vite (обычно http://localhost:5173).

## Сборка

```bash
npm run build
npm run preview
```

## Структура

```
src/
  main.js          — точка входа, состояние и UI
  storage.js       — загрузка/сохранение в localStorage
  widgets.js       — рендер виджетов
  drag.js          — перетаскивание в режиме настроек
  data/defaults.js — начальные данные
  styles/main.css  — стили
```
